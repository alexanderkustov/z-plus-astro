# Zenex Plus (Astro)

This repository contains the Zenex Plus marketing site built with [Astro](https://astro.build/).

## Getting started

```bash
npm install
npm run dev
```

Astro will start the development server at `http://localhost:4321`.

## Project structure

- `src/pages/` — Route-based pages (Home, About, Services, Case Studies).
- `src/layouts/BaseLayout.astro` — Shared layout, navigation, and footer.
- `src/styles/global.css` — Global styles.

## Commands

| Command | Description |
| --- | --- |
| `npm run dev` | Start the local dev server. |
| `npm run build` | Build the production site. |
| `npm run preview` | Preview the production build locally. |
